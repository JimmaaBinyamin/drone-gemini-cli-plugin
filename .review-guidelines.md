# Code Review Guidelines

Please review the code changes and check for:

1. **Security**: SQL injection, XSS, sensitive data exposure
2. **Performance**: N+1 queries, unnecessary allocations, missing caching
3. **Error Handling**: Unchecked errors, missing edge cases
4. **Code Quality**: Naming conventions, DRY principle, readability

Provide a summary with severity levels: CRITICAL, WARNING, INFO.
